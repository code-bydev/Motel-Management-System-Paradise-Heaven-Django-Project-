 🏨 Paradise Heaven – Motel Management System

Paradise Heaven is a Motel Management and Room Booking web application built using Django.
The system allows users to browse available rooms, check facilities, and book rooms through a simple and user-friendly interface.

This project demonstrates full-stack web development using Django, including authentication, dynamic web pages, and booking management.

---

 🚀 Features

--> User Registration and Login
--> Browse Available Rooms
--> Detailed Room Information
--> Room Booking System
--> Payment Summary Page
--> Contact Page
--> User Profile with Avatar
--> Responsive Website Design

 ⚙️ Installation

 1️⃣ Clone the repository

```
git clone https://github.com/your-username/paradise-heaven.git
```

2️⃣ Navigate to the project directory

```
cd paradise-heaven
```

3️⃣ Install required dependencies

```
pip install -r requirements.txt
```

4️⃣ Apply migrations

```
python manage.py migrate
```

5️⃣ Run the development server

```
python manage.py runserver
```

 6️⃣ Open in browser

```
http://127.0.0.1:8000/
```
